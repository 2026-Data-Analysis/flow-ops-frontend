# FlowOps Frontend

React, Vite, and Tailwind CSS frontend for FlowOps.

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```

## Vercel

Use the default Vercel settings from `vercel.json`:

- Build Command: `npm run build`
- Output Directory: `dist`
- Install Command: `npm install`
